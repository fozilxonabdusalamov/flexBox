1. Flexbox Asoslari

Flexbox nima?

Flexbox (Flexible Box Layout) - bu CSS-ning kuchli layouting usuli bo'lib, elementlarni bir o'lchovda (horizontal yoki vertical) joylashtirishga mo'ljallangan.

Asosiy tushunchalar:

  • Flex Container - display: flex qo'yilgan asosiy element
  • Flex Items - flex container ichidagi to'g'ridan-to'g'ri bolalar
  • Main Axis - asosiy yo'nalish (default: left to right)
  • Cross Axis - asosiy yo'nalishga perpendikulyar yo'nalish

5.4 Performance va Best Practices

Performance Tips:

  • Avoid nested flex containers - kerak bo'lgan joylardagina ishlatish
  • Use flex shorthand - flex: 1 instead of flex-grow: 1
  • Prefer flex over float - zamonaviy browserlar uchun
  • Use will-change - animation qilinayotgan flex itemlar uchun

Debugging Tips:

  • Browser DevTools da flex badge larini kuzatish
  • outline: 1px solid red - layout debug uchun
  • Firefox DevTools - eng yaxshi flexbox debugger
  • Chrome DevTools - flex information overlay

Common Mistakes:

  • Min-width/height unutish - content overflow muammolar
  • Flex-basis vs width - flex-basis ni afzal ko'rish
  • Align-content vs align-items - farqni tushunish
  • Margin collapse - flex container da margin collapse bo'lmaydi

Browser Support:

  • Modern Flexbox: IE 11+, Safari 9+, Chrome 29+, Firefox 28+
  • Autoprefixer ishlatish tavsiya etiladi
  • Legacy support uchun fallback strategiyalar

7. CSS Grid Layout - 2D Layouting

Grid Layout nima?

CSS Grid Layout - bu CSS-ning eng kuchli 2D layouting sistemi bo'lib, qatorlar va ustunlar bilan murakkab layoutlar yaratishga mo'ljallangan.

Flexbox vs Grid:

  • Flexbox - 1D (bir yo'nalish) - qator yoki ustun
  • Grid - 2D (ikki yo'nalish) - qator VA ustun
  • Flexbox - content-first approach
  • Grid - layout-first approach

Asosiy tushunchalar:

  • Grid Container - display: grid qo'yilgan asosiy element
  • Grid Items - grid container ichidagi to'g'ridan-to'g'ri bolalar
  • Grid Lines - grid ni bo'ladigan chiziqlar
  • Grid Tracks - qatorlar va ustunlar
  • Grid Cells - eng kichik grid birligi
  • Grid Areas - bir nechta celldan iborat hudud

13.1 Performance Tips

Optimization strategiyalari:

  • Minimal grid complexity - faqat zarur bo'lgan grid lines
  • Efficient gap usage - margin o'rniga gap ishlatish
  • Avoid excessive nesting - deep grid structures oldini olish
  • Use CSS custom properties - grid values uchun variables
  • Prefer fr units - percentage o'rniga fr ishlatish

Debugging Tips:

  • Firefox Grid Inspector - eng yaxshi grid debugging tool
  • Chrome DevTools - grid overlay
  • Grid line names - debug uchun nom berish
  • CSS Grid Generator tools - quick prototyping

Common Mistakes:

  • Overusing positioned items - grid-area o'rniga position absolute
  • Not using semantic HTML - div soup yaratish
  • Ignoring mobile-first - desktop-first design
  • Complex grid-template-areas - o'qib bo'lmas kod

Browser Support:

  • Modern Grid - IE 11 qo'llab-quvvatlamaydi
  • Autoprefixer - legacy browser support
  • Feature queries - @supports ishlatish
  • Progressive enhancement - fallback strategiyalar
